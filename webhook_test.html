📋 노션 버튼 설정 가이드

노션에서 버튼을 만들 때 다음 단계를 따르세요:

  1. 노션 페이지에서 /button 입력
  2. 버튼 이름: "🚀 GIA 아카이브 실행"
  3. 버튼 액션: "Open page" 선택
  4. URL 입력: 위의 웹훅 URL 중 하나를 복사해서 붙여넣기
  5. 새 탭에서 열기 옵션 활성화
💡 추천 URL: 테스트용으로는 "연결 테스트" URL을, 실제 사용시에는 "전체 아카이브 실행" URL을 사용하세요.
